## GCED คืออะไร?

**Global Citizenship Education (GCED)** หรือ **การศึกษาเพื่อความเป็นพลเมืองโลก** คือการศึกษาที่มุ่งบ่มเพาะผู้เรียนให้:

- **เข้าใจโลก** — มองเห็นความเชื่อมโยงระหว่างประเด็นท้องถิ่นและระดับโลก
- **เคารพความแตกต่าง** — ยอมรับความหลากหลายทางวัฒนธรรม ความเชื่อ และมุมมอง
- **คิดวิเคราะห์อย่างมีวิจารณญาณ** — ไม่เชื่อทุกสิ่งที่ได้ยิน ตรวจสอบข้อมูลก่อนตัดสิน
- **ลงมือทำเพื่อสังคมที่ดีขึ้น** — เปลี่ยนความรู้เป็นพฤติกรรมและการกระทำที่สร้างสรรค์

## GCED เชื่อมโยงกับ EIU อย่างไร?

**Education for International Understanding (EIU)** หรือ **การศึกษาเพื่อความเข้าใจอันดีระหว่างประเทศ** เป็นรากฐานสำคัญของ GCED โดยเน้นการสร้าง **วัฒนธรรมแห่งสันติภาพ** ผ่าน:

| ประเด็นหลัก | ตัวอย่างในชีวิตจริง |
|-------------|---------------------|
| สันติภาพ | แก้ความขัดแย้งด้วยสันติวิธี |
| สิทธิมนุษยชน | เคารพศักดิ์ศรีทุกคน ไม่เลือกปฏิบัติ |
| ความหลากหลาย | อยู่ร่วมกับคนต่างวัฒนธรรมอย่างสงบ |
| ความยั่งยืน | ดูแลสิ่งแวดล้อมและทรัพยากรร่วมกัน |
| ความยุติธรรม | มองเห็นความไม่เท่าเทียมและร่วมแก้ไข |

## 3 มิติของพลเมืองโลก (ตามกรอบ UNESCO)

```
        รู้ (Cognitive)
       /            \
  มีค่านิยม          มีทักษะ
 (Socio-emotional)  (Behavioural)
```

1. **รู้** — เข้าใจประเด็นระดับโลก เช่น สิทธิมนุษยชน ความยั่งยืน ความยุติธรรม
2. **มีค่านิยม** — มีความเห็นอกเห็นใจ เคารพความแตกต่าง รู้สึกรับผิดชอบต่อผู้อื่น
3. **มีทักษะ** — สื่อสารอย่างสันติ รู้เท่าทันสื่อ ร่วมมือแก้ปัญหาร่วมกัน

## GCED ในบริบทไทย

GCED ไม่ใช่การ "ทำตามต่างประเทศ" แต่คือการ **นำหลักการสากลมาสร้างความหมายแบบไทย** เช่น

- คุณธรรมและจิตอาสาในวัฒนธรรมไทย
- การอยู่ร่วมกันในสังคมหลากหลาย (ศาสนา ชาติพันธุ์ ภูมิภาค)
- การเคารพผู้ใหญ่ควบคู่กับการเสริมพลังเสียงของเยาวชน
- การเชื่อมโยงกับ **เป้าหมายการพัฒนาที่ยั่งยืน (SDG)** โดยเฉพาะ SDG 4.7

## ใครสามารถเรียนรู้ GCED ได้?

| กลุ่ม | สิ่งที่ได้รับ |
|-------|-------------|
| ครู | แผนการสอน เครื่องมือกิจกรรม แนวประเมินผล |
| ผู้เรียน | บทเรียนสั้น ใบงาน โครงงานลงมือทำ |
| ผู้บริหาร | Policy Brief แนวทางบูรณาการทั้งโรงเรียน |
| ประชาชน | อินโฟกราฟิก เรื่องเล่า ข่าวกิจกรรม |

## เริ่มต้นได้จากอะไร?

1. **มองรอบตัว** — ความหลากหลายในชุมชน โรงเรียน ครอบครัว
2. **ถามก่อนเชื่อ** — ตรวจสอบข่าวก่อนแชร์ต่อ
3. **ฟังเพื่อเข้าใจ** — ไม่ด่วนตัดสินคนที่คิดต่าง
4. **ลงมือทำเล็ก ๆ** — โครงงาน กิจกรรมจิตอาสา การพูดคุยในวงเล็ก
